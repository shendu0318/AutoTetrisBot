Robotic Innovation and Practice

Vision-based Pick-and-Place Robot for Compact Assembly

This project implements automatic block identification, grasping, and compact placement for seven types of Tetris-like pieces using computer vision, hand-eye calibration, inverse kinematics, and genetic algorithm optimization. The full workflow is validated from simulation to the real robot.

Project Overview

Goal: The robot uses a vision system to identify the type, position, and orientation of blocks, plan the grasping order and compact placement, and execute pick-and-place operations automatically.

  • Workflow: Simulation → Real Robot
  • Simulation: Solving inverse kinematics and verifying target tracking via dummy connections
  • Detection: Color segmentation and template matching for pose estimation
  • Calibration: Hand-eye calibration with chessboard pattern
  • Planning: Block legality checking and GA-based optimization for compact layout

Methods and Implementation

1) Detection & Pose Estimation

  • Color thresholding → bounding box center as grasping point
  • Template matching to estimate rotation angle
  • Pose correction for rotated blocks

2) Calibration & Inverse Kinematics

  • Hand-eye calibration using chessboard pattern
  • Analytical IK solver for joint angles
  • Validation with dummy tip-target connections

3) Placement Planning (GA)

  • Coordinate description of block shapes
  • Legality checks: boundary, collision, support
  • Scoring metrics: line removal, transform count, base distance
  • Fitness proportional to value for optimal placement

Results

Both simulation and real-robot experiments confirm successful grasping and placement for blocks with arbitrary rotation angles, including automatic pose correction.
